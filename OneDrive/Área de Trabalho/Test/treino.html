Sobre Git Hub

Commands

git init - (a criação de um repositório estando assim na "brand master")

git commit -m "" (a mensagem do commite usa para comitar, no "" seria o título)

git branch -M "main" (para renomear)

git remote add origin (remote que é o controle, a connection do repositório local com o github)

git push -u origin main (seria um "empurrão" dos commits do repositório local para github)

git checkout -b (criação de uma nova branch, que é uma ramificação(commits se cnidários) da main(linha principal) sem altera-la)

git merge (junção de branch com a main)

git clone (clonar o código do github)

git pull (puxar alterações feitas)

git pull quest (uma solicitação para fazer mudança)

git status (a visualização da pasta)
